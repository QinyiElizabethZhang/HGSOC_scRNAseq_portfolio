# HGSOC single-cell RNA-seq portfolio project

## Overview

This is a beginner/intermediate portfolio project where I practise a basic single-cell RNA-seq workflow in R using Seurat.

The project uses public high-grade serous ovarian cancer scRNA-seq data and is designed to demonstrate my ability to run, document, troubleshoot and interpret a simple cancer bioinformatics workflow.

## Project status

Learning project in progress.

## Planned workflow

1. Set up project structure
2. Load public scRNA-seq count matrix and metadata
3. Create a Seurat object
4. Perform quality control
5. Normalise data
6. Run PCA
7. Generate UMAP
8. Perform clustering
9. Identify marker genes
10. Interpret major cell populations in the context of ovarian cancer

## Skills demonstrated

- Basic Seurat workflow
- scRNA-seq quality control
- Data visualisation
- Marker gene interpretation
- Reproducible documentation
- Beginner-level troubleshooting
